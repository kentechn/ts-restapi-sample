users

Operations about users

Get all users (Admin only)

Retrieve all users with pagination. Only accessible by administrators.

Returns 422 error when:

  • Offset is invalid (negative integer or out of range)
  • Limit parameter is invalid (outside 1-100 range)
  • Offset exceeds total number of available records
Authorizations:
bearerAuth
query Parameters
offset
integer >= 0
Default: 0

Number of records to skip (must be non-negative integer). Returns 422 if the offset is out of range or invalid.

limit
integer [ 1 .. 100 ]
Default: 10

Number of items per page (1-100). Returns 422 if the limit is out of range or invalid.
